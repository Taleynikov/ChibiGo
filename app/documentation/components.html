Обычная кнопка

Обычная ничем не примечательная кнопка.

Кнопка с иконкой до текста

Иконка размещается перед текстом и использует класс модификатор .button_icon_before

Кнопка с иконкой после текста

Иконка размещается после текста и использует класс модификатор .button_icon_after

Кнопка "минимальной ширины"

Кнопка с заданной шириной root * 20, например при размере текста 16px, будет иметь ширину в 320px. Использует класс модификатор .button_opt_minw

Одиночный выбор

Переключатели с возможностью выбора одного пункта

Множественный выбор

Переключатели с возможностью выбора множества пунктов

Заливка

Переключатели с полной заливкой. Использует класс модификатор .toggler_style_fill
